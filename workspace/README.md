# Workspace

This folder is intentionally empty. Put all of your own code here, organized
by track and exercise:

```
workspace/
  track-1-runtime/01-call-stack/solution.js
  track-1-runtime/01-call-stack/NOTES.md
  ...
  track-8-react-fundamentals/   ← your own Vite/HTML app(s)
  track-9-nextjs-fundamentals/  ← your own `create-next-app` project
```

See the root [README.md](../README.md) for the full submission rules.
