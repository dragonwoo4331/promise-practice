# Track 9: Next.js Fundamentals (App Router)

## Overview
이 트랙은 Next.js App Router에 대한 정밀한 멘탈 모델을 구축한다: 중첩 레이아웃과 동적 세그먼트를 가진 파일 시스템 기반 라우팅, Server Component와 Client Component의 경계 및 그 사이를 조합하는 방법, 그리고 Next.js `fetch` 캐싱 시맨틱을 포함한 서버 사이드 데이터 페칭. 코드는 100% 직접 작성해야 한다 — 예제 파일에는 비판하거나, 완성하거나, 고쳐야 할 작은 예시 코드만 포함되어 있으며, 동작하는 완성된 솔루션은 절대 포함되어 있지 않다.

## Before You Start
Exercise 1을 시작하기 전에, `workspace/track-9-nextjs-fundamentals/` 안에서 `npx create-next-app@latest`(App Router 선택, TypeScript는 선택 사항)로 직접 프로젝트를 스캐폴딩하라. 이것은 직접 해야 한다 — AI에게 설정을 대신 맡기지 말고, CLI 프롬프트를 읽고 각 옵션을 이해한 뒤 Enter를 눌러라.
