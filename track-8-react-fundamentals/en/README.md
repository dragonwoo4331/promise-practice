# Track 8: React Fundamentals

## Overview
This track builds a precise mental model of React's core runtime mechanics: state and re-rendering, `useEffect` synchronization with external systems, and modeling asynchronous UI state correctly. Each exercise requires you to predict behavior before running code, then reconcile your prediction against what you actually observe. You are expected to write 100% of the code yourself — the exercise files contain only small illustrative snippets to critique, complete, or fix, never a working solution.

## Exercises
1. [State & the Re-render Mental Model](./01-state-and-rerender.md) — Easy
2. [useEffect Dependency Array, Cleanup, and the Async Race Condition](./02-useeffect-deps-cleanup.md) — Medium
3. [Loading / Error / Data State Pattern](./03-loading-error-data-state.md) — Medium
