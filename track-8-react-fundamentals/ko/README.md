# Track 8: React Fundamentals

## Overview
이 트랙은 React의 핵심 런타임 동작에 대한 정밀한 멘탈 모델을 구축한다: State와 Re-render, 외부 시스템과의 동기화를 위한 `useEffect`, 그리고 비동기 UI 상태를 올바르게 모델링하는 방법. 각 예제는 코드를 실행하기 전에 동작을 예측한 뒤, 실제로 관찰한 내용과 예측을 대조하도록 요구한다. 코드는 100% 직접 작성해야 한다 — 예제 파일에는 비판하거나, 완성하거나, 고쳐야 할 작은 예시 코드만 포함되어 있으며, 동작하는 완성된 솔루션은 절대 포함되어 있지 않다.
